### Prerequisites

- .NET 8.0 or later

### Installing

1. Clone the repository:
    ````bash
    git clone https://github.com/yourusername/yourrepository.git
    ````
2. Navigate to the project directory:
    ````bash
    cd yourrepository
    ````
3. Build the project:
    ````bash
    dotnet build
    ````

### Usage

Run the project with the following command:

````bash
dotnet run
````
